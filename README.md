# SPORT FLOW 

Este proyecto consiste en el diseño y desarrollo de una **página web deportiva interactiva**, donde toda la navegación ocurre dentro de **una sola pantalla**, permitiendo una experiencia fluida y dinámica para el usuario.

El objetivo del proyecto fue crear una interfaz clara, visualmente atractiva y fácil de navegar, implementando **interacciones dinámicas con JavaScript** para mejorar la experiencia del usuario.

---

## Objetivo del proyecto

Diseñar una experiencia web que permita al usuario explorar contenido deportivo de forma rápida e intuitiva, evitando recargas de página y facilitando la navegación dentro de una misma vista.

El proyecto integra **diseño de interfaz y desarrollo Front-End**, buscando equilibrio entre funcionalidad, estética y usabilidad.

---

## Metodología

Durante el desarrollo del proyecto se consideraron principios de **UX/UI**, enfocándose en:

- Organización clara de la información
- Navegación intuitiva
- Interacciones dinámicas
- Diseño visual enfocado en contenido deportivo

---

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Bootstrap**

---

## Funcionalidades

- Navegación en **una sola pantalla**
- **Interacciones dinámicas con JavaScript**
- Diseño **responsive**
- Componentes visuales organizados para facilitar la exploración del contenido
- Estructura modular para facilitar futuras mejoras
